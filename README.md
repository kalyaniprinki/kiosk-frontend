# Kiosk Frontend (React + Vite)
Minimal frontend for a printing kiosk with three login types: user, admin, kiosk.

## Run locally
1. `npm install`
2. `npm run dev`
3. Open http://localhost:5173

This is a minimal scaffold: the API calls are mocked inside `src/api/mockApi.js`. Replace with your real API.
